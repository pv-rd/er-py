# ErPy - Error data processing for Python 

## Purposes

The module can make it easier to 
- read measured physical quantities from files, 
- calculate indirect measurements, and 
- visualize identified patterns. 

> The module can be used to compile reports on laboratory work.

## Structure

### 1. class Value
It implements a data type used by the module for keeping and working values with their errors.
### 2. class Data
It is needed to implement an array of values of our data type.
It use **NumPy** module.
### 3. class Plot
It is needed to visualize data arrays and approximate.
It use **Matplotlib** module.
